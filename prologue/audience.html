ii. 读者

你要读这本书可能会有很多原因。很多人问的问题之一是：“为什么本来只要下载安装一个存在的 Linux 系统就行却非要克服所有困难去手工编制一个？”

这个工程存在的一个重要原因就是要帮你由内而外学习 Linux 系统是如何工作的。编制一套 LFS 系统帮助了解是什么使 Linux 工作，这些东西如何协同工作并相互依赖。可以证明，这种学习经历能提供的最好的东西之一就是定制一个 Linux 系统来满足你自己独特需求的能力。

LFS 的另一个关键的好处就是允许你更多地控制系统而不必依赖别人的 Linux 实现。有了 LFS，你坐在驾驶位上来支配系统的每个层面。

LFS 允许你创建非常紧凑的 Linux 系统。安装常规发行版的时候，你经常要被迫安装大量可能永远不会被理解和使用的程序。这些程序浪费资源。你可能要反驳说有当今的硬盘和 CPU，这样的资源不必再考虑了。然而有的时候，你却至少仍被尺寸限制着。想一想可引导 CD，优盘和嵌入式系统。这都是 LFS 可以有利的地方。

定制 Linux 系统的另一个优势就是安全。通过从源代码编译整个系统，你能够审查一切并能应用所有需要的安全补丁。不用再等别人来编译二进制程序包以修复安全漏洞。除非你自己检查和实现补丁，你无法绝对保证新的二进制程序正确编制并能精确地修复问题。

Linux From Scratch 的目标是编制一个完整并且可用的底层系统。你如果没想编制自己的 LFS 系统，可能就无法从本书的信息中获益。

编制自己的 LFS 系统还有许多好的原因，在此就不意一一列举了。说到底，教育才是最关键的因素。当你继续你的 LFS 经历的时候，你就会明白信息和知识带来的力量究竟有多大了。
