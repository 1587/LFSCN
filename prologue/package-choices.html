v. 对本书中程序包的解释

前面提到过，LFS 的目的是构建一个完整可用的基础系统。这包括一些能够复制自身并提供相对最小的工作环境的软件包，用户在此之上定制自己的系统。这并不意味着LFS是最小的操作系统。几个重要的软件包在其他系统上并不严格需求。下面列出了对本书中每个程序包的解释。

  • Autoconf

    此程序包包含用于根据开发者的模板生成可以自动配置源代码的 shell 脚本的程序。当编制过成更新的时候通常要用它重新编制程序包。

  • Automake

    此程序包包含用于根据模板生成 Make 文件的程序。当编制过成更新的时候通常要用它重新编制程序包。

  • Bash

    此程序包满足一项 LSB 核心需求，给系统提供了一个 Bourne Shell 界面。因为使用广泛和对基本 shell 功能以外的扩展兼容性良好而选用。

  • Binutils

    此程序包包含链接器、汇编器和处理对象文件的其他工具。其中的程序用于编译 LFS 系统内外的大部分程序包。

  • Bison

    此程序包包含 GNU 版本的 yacc (又一个编译器代码生成器)，用于编制其他几个 LFS 系统中的程序。

  • Bzip2

    此程序包包含用于压缩和解压缩文件的程序，用于解压许多 LFS 程序包。

  • Coreutils

    此程序包包含大量用于查看和操作文件和目录的必要工具。这些程序用于命令行文件管理，被 LFS 中每个程序包的安装过程需求。

  • DejaGNU

    此程序包包含一个用于测试其他程序的框架，只安装于临时工具链中。

  • Diffutils

    此程序包包含用于显示文件和文件夹差异的程序。这些程序可以用于生成补丁，也用在许多程序包编制过程中。

  • Expect

    此程序包包含一个用于和其他交互程序进行脚本指导对话的程序，广泛用于测试其他程序包。只安装于临时工具链中。

  • E2fsprogs

    此程序包包含用于处理 ext2、ext3 和 ext4 文件系统的工具。这些是 Linux支持的最广泛使用并且经过彻底测试的文件系统。

  • File

    此程序包包含一个用于检测文件类型的工具，被几个程序包的编制需求。

  • Findutils

    此程序包包含用于在文件系统中查找文件的工具，被许多程序包的编制脚本使用。

  • Flex

    此程序包包含一个用于生成识别文本样式的程序的工具，是 GNU 版本的 lex (词汇分析器)，被用于几个程序包的编制需求。

  • Gawk

    此程序包包含用于操作文本文件的程序，是 GNU 版本的 awk (Aho-Weinberg-Kernighan)，被很多其他程序包的编制脚本使用。

  • Gcc

    此程序包为 GNU 编译器集，包含 C 和 C++，还有几个 LFS 不编制的编译器。

  • GDBM

    此程序包包含 GNU 数据库管理库，只被 Man-DB 需求。

  • Gettext

    此程序包包含用于很多程序国际化和本地化的工具和库。

  • Glibc

    此程序包包含标准 C 库，Linux 程序没有它无法工作。

  • GMP

    此程序包包含数学库，提供任意精度算术运算的常用函数，用于编制 Gcc。

  • Grep

    此程序包含用于在文件内容中查找的工具。这些程序在多数程序包的编制脚本中使用。

  • Groff

    此程序包包含用于处理和格式化文本的程序，一个重要的功能是格式化手册页。

  • GRUB

    此程序包为通用引导加载器。它是几种可用的引导加载器中最灵活的一种。

  • Gzip

    此程序包包含用于压缩和解压缩文件的程序。用于解压缩 LFS 内外的很多软件包。

  • Iana-etc

    此程序给网络服务和协议提供数据，用于启用正确的网络兼容性。

  • Inetutils

    此程序包包含用于基本网络管理的程序。

  • IProute2

    此程序包包含用于基本和高级的 IPv4 和 IPv6 网络的程序，因为 IPv6 兼容性被选用

  • Kbd

    此程序包包含键盘映射表文件和用于非标准 US 键盘的键盘工具，还有一些控制台字体。

  • Kmod

    此程序包包含用于管理 Linux 内核模块的程序。

  • Less

    此程序包包含一个非常不错的文本文件查看器，允许在浏览文件时上下卷页，也被 Man-DB 用于查看手册页。

  • Libtool

    此程序包包含 GNU 通用库支持脚本，把使用共享库的方式以固定而且可移植的方式打包。它被几个其他 LFS 程序包的测试套件需求。

  • Linux Kernel

    此程序包为操作系统内核，即 GNU/Linux 环境的 Linux。

  • M4

    此程序包包含一个通用文本宏处理器，用作其他程序的编制工具。

  • Make

    此程序包包含一个指导程序包编制的程序，被 LFS 中几乎每个程序包需求。

  • Man-DB

    此程序包包含用于查找和查看手册页的工具。使用它而不是 man 程序包是由于更好的国际化兼容性。它支持 man 程序。

  • Man-pages

    此程序包包含基本 Linux 手册页的实际内容。

  • MPC

    此程序包包含用于复数算术运算的函数，被 Gcc 需求。

  • MPFR

    此程序包包含用于多精度算术的函数，被 Gcc 需求。

  • Ncurses

    此程序包包含用于不依赖终端处理字符屏幕的库。它经常用于给菜单系统提供光标控制，被 LFS 系统中的很多程序包需求。

  • Patch

    此程序包包含一个根据由 diff 程序创建的 补丁 修改或创建文件的程序，被几个 LFS 程序包的编制过成需求。

  • Perl

    此程序包包含解释型语言 PERL 的解释器，被几个 LFS 程序包的安装和测试套件需求。

  • Procps

    此程序包包含用于监视进程的工具。这些工具对系统管理非常有用，并被 LFS 系统启动脚本使用。

  • Psmisc

    此程序包包含用于显示运行中进程信息的程序，对系统管理非常有用。

  • Readline

    此程序包为一组库，提供命令行编辑和历史记录功能，被 Bash 使用。

  • Sed

    此程序包允许不打开文本编辑器就编辑文本，也被多数 LFS 程序包的配置脚本需求。

  • Shadow

    包含用于以安全方式处理密码的程序。

  • Sysklogd

    此程序包包含用于记录系统日志，例如由内核或 daemon 进程在不寻常事件发生时给出的消息的程序。

  • Sysvinit

    此程序包提供 init 程序。该进程是 Linux 系统中所有进程的父进程。

  • Tar

    此程序包为 LFS 中用到的所有程序包虚拟提供归档和提取兼容性。

  • Tcl

    此程序包包含 LFS 很多程序包测试套件用到的工具命令语言。只安装于临时工具链。

  • Texinfo

    此程序包包含用于信息页读写、转换的程序，在许多 LFS 程序包的安装进程中用到。

  • Udev

    此程序包包含用于动态创建设备节点的程序，以免在 /dev 目录中创建数以千记的静态设备。

  • Util-linux

    此程序包包含大量工具程序。其中包含用于处理文件系统、控制台、分区和消息的工具。

  • Vim

    此程序包包含一个编辑器，因为兼容传统的 vi 编辑器并拥有强大的兼容性而选用。编辑器是个人选择。可换成其他编辑器。

  • XZ Utils

    此程序包包含用于压缩和解压缩文件的程序。它提供通常可用的最大压缩率并可用于解压 XZ 或 LZMA 格式的包。

  • Zlib

    此程序包包含一些程序用到的压缩和解压缩途径。
