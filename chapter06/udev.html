6.61. Udev-195 (提取自 systemd-195)

Udev 程序包包含用于动态创建设备节点的程序。Udev 的开发已经融合到 systemd 中，但由于其大部分与 LFS 不兼容，我们此处只编译安装需要的 udev 文件。

预计编制时间：0.1 SBU
需求磁盘空间：18 MB

6.61.1. 安装 Udev

注意

此程序包与其他程序包略有不同，虽然安装的应用程序是 udev 但是初始解压缩的程序包是 systemd-195.tar.xz。在切换至 systemd 目录中之后按以下指令执行。

udev-lfs 压缩包包含 LFS 特有的文件，用于编制 Udev。将其解包至 systemd 原地码目录中： 

tar -xvf ../udev-lfs-195.tar.bz2

编译程序包： 

make -f udev-lfs-195/Makefile.lfs

安装程序包： 

make -f udev-lfs-195/Makefile.lfs install

最后设置稳定的 udev 网络规则。此任务在第 7.2.1 节 “创建网络接口的稳定名称”中具体讲解。注意 /sys 和 /proc 文件系统必须按本章开始处说明的方式挂载于 chroot 环境中，以下脚本才能够工作。 

bash udev-lfs-195/init-net-rules.sh

6.61.2. Udev 的内容

安装的程序：accelerometer、ata_id、cdrom_id、collect、mtd_probe、scsi_id、v4l_id、udevadm 和 udevd
安装的库：libudev.so
安装的目录：/etc/udev、/lib/udev、/lib/firmware、/usr/share/doc/udev

简介

ata_id

对于 ATA 驱动器，给 Udev 提供一个独有的字符串以及额外信息 (uuid 和卷标)

cdrom_id

使 Udev 兼容 CD-ROM 或者 DVD-ROM 驱动器

collect

给定当前以及一列 uevent ID，注册当前 ID 并指示所有目标 ID 是否注册过

scsi_id

基于特定设备对 SCSI INQUIRY 命令返回的数据给 Udev 提供一个独有的 SCSI 标识符

udevadm

通用 udev 管理工具：控制 udevd daemon、从 Udev 数据库中提供信息、监视 uevent、并对于给定的设备激发uevent

udevd

一个 daemon，监听 netlink 接口上的 uevent、创建设备并运行配置好的外置程序响应这些 uevent

libudev

udev 设备信息的一个库界面

/etc/udev

包含 Udev 配置文件、设备权限和设备命名规则
