6.14. GMP-5.0.5

GMP 程序包包含数学库。其中包含用于任意精度算数的实用函数。

预计编制时间：1.2 SBU
需求磁盘空间：50 MB

6.14.1. 安装 GMP

注意

如果你想要编制 32 位 x86 系统, 却有支持 64 位代码运行的 CPU 且环境中有指定的 CFLAGS 存在, 配置脚本将会尝试配置 64 位并失败。要避免该错误，调用以下配置命令 

ABI=32 ./configure ...

准备编译 GMP： 

./configure --prefix=/usr --enable-cxx --enable-mpbsd

新的配置选项的含义：

--enable-cxx

此变量使 C++ 支持有效

--enable-mpbsd

此项编制 Berkeley MP 兼容库

编译程序包： 

make

重要

本节中 GMP 的测试套件被是关十分重要的，无论如何都不要跳过。

测试结果： 

make check 2>&1 | tee gmp-check-log

确保套件中全部 166 个测试都通过。要确认结果，执行以下命令： 

awk '/tests passed/{total+=$2} ; END{print total}' gmp-check-log

安装程序包： 

make install

如果需要，安装文档： 

mkdir -v /usr/share/doc/gmp-5.0.5
cp    -v doc/{isa_abi_headache,configuration} doc/*.html \
         /usr/share/doc/gmp-5.0.5

6.14.2. GMP 的内容

安装的库：libgmp.{a,so}, libgmpxx.{a,so} 和 libmp.{a,so}
安装的目录：/usr/share/doc/gmp-5.0.5

简介

libgmp

包含精确算术函数。

libgmpxx

包含 C++ 精确算术函数。

libmp

包含 Berkeley MP 算术函数。
