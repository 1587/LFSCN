6.36. Inetutils-1.9.1

Inetutils 程序包包含用于基本网络的程序。

预计编制时间: 0.4 SBU
需求磁盘空间: 27 MB

6.36.1. 安装 Inetutils

修复此程序包与 Glibc-2.16.0 不兼容的问题： 

sed -i -e '/gets is a/d' lib/stdio.in.h

准备编译 Inetutils： 

./configure --prefix=/usr  \
    --libexecdir=/usr/sbin \
    --localstatedir=/var   \
    --disable-ifconfig     \
    --disable-logger       \
    --disable-syslogd      \
    --disable-whois        \
    --disable-servers

配置选项的含义：

--disable-ifconfig

此许想阻止 Inetutils 安装 ifconfig 程序，此程序用于配置网络接口。LFS 使用 IPRoute2 中的 ip 完成此任务。

--disable-logger

此选项防止 Inetutils 安装 logger 程序，它被脚本用于给系统日志 Daemon 传递消息。Util-linux 此前已经安装过了。

--disable-syslogd

此选项阻止 Inetutils 安装系统日志 Daemon，它已经被 Sysklogd 程序包安装过了。

--disable-whois

此选项禁用过时的 whois 客户端。配置一个更好的 whois 客户端的指令位于 BLFS 中。

--disable-servers

此项禁用多种网络服务器。这些服务器在基本 LFS 系统中并不合适。其中有的天生不安全，有的只在信任网络内安全。更多信息位于 http://www.linuxfromscratch.org/blfs/view/svn/basicnet/inetutils.html。注意其中很多服务器都有更好的替代品。

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install
make -C doc html
make -C doc install-html docdir=/usr/share/doc/inetutils-1.9.1

将某些程序移动到与 FHS 兼容的位置： 

mv -v /usr/bin/{hostname,ping,ping6} /bin
mv -v /usr/bin/traceroute /sbin

6.36.2. Inetutils 的内容

安装的程序: ftp、hostname、ping、ping6、rcp、rexec、rlogin、rsh、talk、telnet、tftp 和 traceroute

简介

ftp

文件传输程序

hostname

报告或者设置主机名

ping

发送要求回执的数据包并报告收到回执所用的时间

ping6

ping 的用于 IPv6 网络的版本

rcp

进行远程文件复制

rexec

在远程主机上运行指令

rlogin

进行远程登录

rsh

运行远程 shell

talk

用于与其他用户聊天

telnet

TELNET 协议的一个界面

tftp

简易文件传输程序

traceroute

跟踪你正在工作的主机到网络上另一台主机的数据包通过的路由，显示沿途所有中介跃点 (网关)
