6.16. MPC-1.0.1

MPC 程序包包含用于复数算数的库，精度高而且可变，结果也会被正确近似。

预计编制时间：0.4 SBU
需求磁盘空间：10.2 MB

6.16.1. 安装 MPC

准备编译 MPC： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.16.2. MPC 的内容

安装的库：libmpc.{a,so}

简介

libmpc

包含复数数学函数
