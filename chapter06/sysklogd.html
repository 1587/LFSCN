6.57. Sysklogd-1.5

Syslogd 程序包包含用于记录系统信息的程序，比如非正常事件发生时由内核给出的信息。

预计编制时间：少于 0.1 SBU
需求磁盘空间：0.6 MB

6.57.1. 安装 Sysklogd

编译程序包： 

make

这个程序包不包含测试套件。

安装程序包： 

make BINDIR=/sbin install

6.57.2. 配置 Sysklogd

运行一下命令来创建一个新的 /etc/syslog.conf 文件： 

cat > /etc/syslog.conf << "EOF"
# Begin /etc/syslog.conf

auth,authpriv.* -/var/log/auth.log
*.*;auth,authpriv.none -/var/log/sys.log
daemon.* -/var/log/daemon.log
kern.* -/var/log/kern.log
mail.* -/var/log/mail.log
user.* -/var/log/user.log
*.emerg *

# End /etc/syslog.conf
EOF

6.57.3. Sysklogd 的内容

安装的程序：klogd 和 syslogd

简介

klogd

一个用以拦截和记录内核信息的系统 daemon

syslogd

记录系统程序为日志记录提供的信息。每个记录的信息包含至少一个时间戳和一个用户名，一般还包括程序名，但取决于记录 daemon 的可靠程度
