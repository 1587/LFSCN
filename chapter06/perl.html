6.37. Perl-5.16.1

Perl 程序包包含 Perl 语言的相关工具。

预计编制时间：7.5 SBU
需求磁盘空间：247 MB

6.37.1. 安装 Perl

首先创建一个基本的 /etc/hosts 文件供 Perl 的配置文件和可选的测试套件使用： 

echo "127.0.0.1 localhost $(hostname)" > /etc/hosts

此版本的 Perl 现在编制 Compress::Raw::Zlib 模块。默认地，Perl 会使用其内置的 Zlib 源代码副本。运行以下命令使 Perl 使用系统上安装的 Zlib 库： 

sed -i -e "s|BUILD_ZLIB\s*= True|BUILD_ZLIB = False|"           \
       -e "s|INCLUDE\s*= ./zlib-src|INCLUDE    = /usr/include|" \
       -e "s|LIB\s*= ./zlib-src|LIB        = /usr/lib|"         \
    cpan/Compress-Raw-Zlib/config.in

要完全控制 Perl 设置的方式，你可以移除以下命令中的 “-des” 选项并手工选择该程序包编制的方式。也可以直接使用以下命令使用 Perl 自动检测到的默认值： 

sh Configure -des -Dprefix=/usr                 \
                  -Dvendorprefix=/usr           \
                  -Dman1dir=/usr/share/man/man1 \
                  -Dman3dir=/usr/share/man/man3 \
                  -Dpager="/usr/bin/less -isR"  \
                  -Duseshrplib

配置脚本的含义：

-Dvendorprefix=/usr

此项确保 perl 知道如何告诉程序包将 perl 模块安装到什么地方。

-Dpager="/usr/bin/less -isR"

此项修复 perldoc 调用 less 程序的方法问题。

-Dman1dir=/usr/share/man/man1 -Dman3dir=/usr/share/man/man3

因为 Groff 还没有安装，Configure 认为我们不需要 Perl 的手册页。执行以下会覆盖这个决定。

-Duseshrplib

编制某些 perl 模块需要的共享的 libperl。

编译程序包： 

make

要测试结果 (大约 2.5 SBU)，执行： 

make -k test

安装程序包： 

make install

6.37.2. Perl 的内容

安装的程序：a2p、c2ph、config_data、corelist、cpan、cpan2dist、cpanp cpanp-run-perl、enc2xs、find2perl、h2ph、h2xs instmodsh、json_pp、libnetcfg、perl、perl5.16.1 (到 perl 的链接)、perlbug、perldoc、perlivp、perlthanks (link to perlbug)、piconv、pl2pm、pod2html、pod2latex、pod2man、pod2text、pod2usage、podchecker、podselect、prove、psed (link to s2p)、pstruct (link to c2ph)、ptar、ptardiff、ptargrep、s2p、shasum、splain、xsubpp 和 zipdetails
安装的库：数百个，此处无法一一列出
安装的目录：/usr/lib/perl5

简介

a2p

将 awk 翻译成 Perl

c2ph

转储 cc -g -S 创建的 C 结构

config_data

询问或者修改 Perl 模块的配置

corelist

Module::CoreList 的命令行前端

cpan

从命令行与 CPAN (Perl 综合典藏网) 交互

cpan2dist

CPANPLUS 发行版创建器

cpanp

CPANPLUS 启动器

cpanp-run-perl

Perl 脚本，用于启用在生成的进程的每次写入操作之后立即冲掉输出缓存

enc2xs

从 Unicode 字符映射表或者 Tcl 映射表给编码模块编制一个 Perl 扩展

find2perl

将 find 命令翻译成 Perl

h2ph

将 .h C 头文件转换成 .ph Perl 头文件

h2xs

将 .h C 头文件转换成 Perl 扩展

instmodsh

Shell 脚本，用于检查安装的 Perl 模块，还可以用于将以安装的模块打包

json_pp

在特定的输入输出格式之间转换数据

libnetcfg

可以用于配置 libnet Perl 模块

perl

将 C、sed、awk 和 sh 的某些最好的特性组合成一种像瑞士军刀一样的语言

perl5.16.1

到 perl 的硬链接

perlbug

用于创建有关 Perl 或其包含的模块的漏洞报告并发送之

perldoc

以 pod 格式显示一份嵌入在 Perl 安装树或脚本中的文档

perlivp

Perl 安装验证过程；可以用于验证 Perl 及其库是否正确安装

perlthanks

用于创建一条感谢消息供发给 Perl 开发者

piconv

字符编码转换器 iconv 的一个 Perl 版本

pl2pm

一个粗略的工具，将 Perl4 的 .pl 文件转换成 Perl5 的 .pm 模块

pod2html

将 pod 格式的文件转换成 HTML 格式

pod2latex

将 pod 格式的文件转换为 LaTeX 格式

pod2man

将 pod 数据转换成格式化的 *roff 数据

pod2text

将 pod 数据抓换成格式化的 ASCII 文本

pod2usage

显示文件中嵌入的 pod 文档中的使用方式

podchecker

检查 pod 格式文档的语法

podselect

显示 pod 文档的选定部分

prove

命令行工具，用于依据 Test::Harness 模块运行测试

psed

流编辑器 sed 的 Perl 版本

pstruct

转储 cc -g -S 尝试产生的 C 结构

ptar

一个以 Perl 语言写成的类似 tar 的程序

ptardiff

一个 Perl 程序，将提取的归档文件与未归档的进行比较

ptargrep

一个 Perl 程序，对 tar 归档中的文件内同进行匹配

s2p

将 sed 脚本翻译成 Perl

shasum

打印或者检查 SHA 指纹码

splain

用于强制详细输出 Perl 诊断的警告

xsubpp

将 Perl XS 代码转换成 C 代码

zipdetails

显示一 Zip 文件的内在结构细节
