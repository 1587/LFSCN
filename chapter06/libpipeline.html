6.53. Libpipeline-1.2.2

Libpipeline 程序包包含一个以灵活便捷方式修改子进程管线的库。

预计编制时间：0.2 SBU
需求磁盘空间：7.4 MB

6.53.1. 安装 Libpipeline

准备编译 Libpipeline： 

PKG_CONFIG_PATH=/tools/lib/pkgconfig ./configure --prefix=/usr

配置脚本的含义：

PKG_CONFIG_PATH

使用 pkg-config 获取在第 5.13 节 “Check-0.9.9”中编制的测试库元数据。

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.53.2. Libpipeline 的内容

安装的库：libpipeline.so

简介

libpipeline

此库用于在子进程之间安全建立管线
