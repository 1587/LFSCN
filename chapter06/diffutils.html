6.40. Diffutils-3.2

Diffutils 程序包包含用于显示文件或目录之间差异的程序。

预计编制时间：0.5 SBU
需求磁盘空间：25 MB

6.40.1. 安装 Diffutils

修复此程序包与 Glibc-2.16.0 不兼容的问题： 

sed -i -e '/gets is a/d' lib/stdio.in.h

准备编译 Diffutils： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.40.2. Diffutils 的内容

安装的程序：cmp, diff, diff3 和 sdiff

简介

cmp

比较两个文本文件，按字节输出差异

diff

比较两个文本文件，按行输出差异

diff3

逐行比较三个文件

sdiff

融合两个文件，交互输出结果
