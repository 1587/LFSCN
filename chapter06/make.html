6.54. Make-3.82

Make 程序包包含一个用于编译程序包的程序。

预计编制时间：0.4 SBU
需求磁盘空间：11.3 MB

6.54.1. 安装 Make

首先应用一些上游补丁： 

patch -Np1 -i ../make-3.82-upstream_fixes-3.patch

准备编译 Make： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.54.2. Make 的内容

安装的程序：make

简介

make

自动获取程序包需要 (重新) 编译的部分并执行对应的命令
