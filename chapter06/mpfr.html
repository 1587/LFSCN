6.15. MPFR-3.1.1

MPFR 程序包包含用于多精度数学的函数。

预计编制时间：0.8 SBU
需求磁盘空间：27 MB

6.15.1. 安装 MPFR

准备编译 MPFR： 

./configure  --prefix=/usr        \
             --enable-thread-safe \
             --docdir=/usr/share/doc/mpfr-3.1.1

编译程序包： 

make

重要

本节中 MPFR 的测试套件是十分重要的，任何情况下都不要跳过。

测试结果，确保所有测试通过： 

make check

安装程序包： 

make install

安装文档： 

make html
make install-html

6.15.2. MPFR 的内容

安装的库：libmpfr.{a,so}
安装的目录：/usr/share/doc/mpfr-3.1.1

简介

libmpfr

包含多精度数学函数。
