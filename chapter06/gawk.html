6.41. Gawk-4.0.1

Gawk 程序包包含用于操作文本文件的程序。

预计编制时间：0.2 SBU
需求磁盘空间：30 MB

6.41.1. 安装 Gawk

准备编译 Gawk： 

./configure --prefix=/usr --libexecdir=/usr/lib

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

如果需要，安装文档： 

mkdir -v /usr/share/doc/gawk-4.0.1
cp    -v doc/{awkforai.txt,*.{eps,pdf,jpg}} /usr/share/doc/gawk-4.0.1

6.41.2. Gawk 的内容

安装的程序：awk (到 gawk 的链接), dgawk, gawk, gawk-4.0.1, grcat, igawk, pgawk, pgawk-4.0.1 和 pwcat
安装的目录：/usr/lib/awk, /usr/share/awk

简介

awk

到 gawk 的链接

dgawk

awk 调试器

gawk

一个用于修改文本文件的程序；是 GNU 实现的 awk

gawk-4.0.1

到 gawk 的硬链接

grcat

转储组数据库 /etc/group

igawk

给 gawk 提供包含文件的能力

pgawk

gawk 的带分析版本

pgawk-4.0.1

到 pgawk 的硬链接

pwcat

转储密码数据库 /etc/passwd
