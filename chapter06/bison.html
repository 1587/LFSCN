6.29. Bison-2.6.2

Bison 程序包包含分析器生成器

预计编制时间: 1.3 SBU
需求磁盘空间: 34 MB

6.29.1. 安装 Bison

准备编译 Bison： 

./configure --prefix=/usr

如果 $PATH 中已经有一个 bison，配置会导致新编译的副本不支持国际化。以下添加将会修复此问题： 

echo '#define YYENABLE_NLS 1' >> lib/config.h

编译程序包： 

make

要测试结果 (大约 0.5 SBU)，执行： 

make check

安装程序包： 

make install

6.29.2. Bison 的内容

安装的程序: bison 和 yacc
安装的库: liby.a
安装的目录: /usr/share/bison

简介

bison

以一系列规则创建用于分析文本结构的程序；是 Yacc 的替代

yacc

bison 的包裹，针对仍然调用 yacc 而不是 bison 的程序；它调用 bison 并传递 -y 选项

liby.a

Yacc 库包含与 Yacc 兼容的 yyerror 和 main 函数的实现；此库通常不会很常用，但 POSIX 要求其存在
