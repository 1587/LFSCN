6.60. Texinfo-4.13a

Texinfo 程序包包含用于读写和转码信息页面的程序。

预计编制时间：0.2 SBU
需求磁盘空间：24 MB

6.60.1. 安装 Texinfo

准备编译 Texinfo： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果，执行 

make check

安装程序包： 

make install

可以选择安装 TeX 的某些组件： 

make TEXMF=/usr/share/texmf install-tex

make 参数的含义：

TEXMF=/usr/share/texmf

若此后安装 TeX 程序包，TEXMF 这个 makefile 变量将保存 TeX 目录树根的位置。

Info 文档系统使用纯文本文件保存菜单条目。此文件位于 /usr/share/info/dir。然而由于不同程序的 Makefile 有时会出现问题，此文件可能会与系统上安装的信息页不同步。以下可选命令可以重新创建此文件： 

cd /usr/share/info
rm -v dir
for f in *
do install-info $f dir 2>/dev/null
done

6.60.2. Texinfo 的内容

安装的程序：info、infokey、install-info、makeinfo、pdftexi2dvi、texi2dvi、texi2pdf 和 texindex
安装的目录：/usr/share/texinfo

简介

info

用于阅读类似手册页的信息页，通常比只是解释所有可用的命令行选项更加深入；例如 man bison 和 info bison 明显不同

infokey

将包含 Info 自定义的源文件编译成二进制格式

install-info

用于安装信息页；它更新 info 索引文件

makeinfo

将给定的 Texinfo 源代码文档翻译成信息页、纯文本或者 HTML

pdftexi2dvi

用于将给定的 Texinfo 文档格式化为 PDF 文件

texi2dvi

用于将给定的 Texinfo 文档格式化为 dvi 文件

texi2pdf

用于将给定的 Texinfo 文档格式化为 PDF 文件

texindex

用于排序 Texinfo 索引文件
