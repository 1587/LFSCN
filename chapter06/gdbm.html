6.35. GDBM-1.10

GDBM 程序包包含 GNU 数据库管理器。这是一个磁盘文件格式数据库，将键值/数据对存储在单一文件中。实际上存储的任何记录的实际数据都由不同的键值索引，比在文本文件中访问更快。

预计编制时间：0.1 SBU
需求磁盘空间：8.5 MB

6.35.1. 安装 GDBM

准备编译 GDBM： 

./configure --prefix=/usr --enable-libgdbm-compat

配置选项的含义：

--enable-libgdbm-compat

此开关启用编制 libgdbm 兼容库，因为 LFS 以外的某些程序包需要它提供旧式的 DBM 途径。

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.35.2. GDBM 的内容

安装的程序：testgdbm
安装的库：libgdbm.{so,a} 和 libgdbm_compat.{so,a}

简介

testgdbm

检测并修改 GDBM 数据库

libgdbm

包含用来修改哈希过的数据库的函数
