6.28. M4-1.4.16

M4 程序包包含一个宏处理器。

预计编制时间：0.4 SBU
需求磁盘空间：26.6 MB

6.28.1. 安装 M4

修复此程序包和 Glibc-2.16.0 不兼容的问题： 

sed -i -e '/gets is a/d' lib/stdio.in.h

准备编译 M4： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果，修复一个测试程序然后运行： 

sed -i -e '41s/ENOENT/& || errno == EINVAL/' tests/test-readlink.h
make check

安装程序包： 

make install

6.28.2. M4 的内容

安装的程序：m4

简介

m4

复制给定的文件并展开其中的宏。这些宏可以是内建的也可以是用户定义的，可以包含任意数量的参数。初次之外 m4 还有内建的包含文件、运行 Unix 命令、进行整数运算、修改文本、递归等功能。m4 程序可以用作编译器前端或者独立的宏处理器。
