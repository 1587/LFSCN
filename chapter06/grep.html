6.31. Grep-2.14

Grep 程序包包含用于搜索文件内容的程序。

预计编制时间：0.4 SBU
需求磁盘空间：30 MB

6.31.1. 安装 Grep

准备编译 Grep： 

./configure --prefix=/usr --bindir=/bin

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.31.2. Grep 的内容

安装的程序：egrep、fgrep 和 grep

简介

egrep

打印匹配指定扩展正则表达式的行

fgrep

打印匹配指定字符串的行

grep

打印匹配基本正则表达式的行
