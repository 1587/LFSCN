6.55. Man-DB-2.6.3

Man-DB 程序包包含用于寻找和查看手册页的程序：

预计编制时间：0.5 SBU
需求磁盘空间：27 MB

6.55.1. 安装 Man-DB

准备编译 Man-DB： 

./configure --prefix=/usr                        \
            --libexecdir=/usr/lib                \
            --docdir=/usr/share/doc/man-db-2.6.3 \
            --sysconfdir=/etc                    \
            --disable-setuid                     \
            --with-browser=/usr/bin/lynx         \
            --with-vgrind=/usr/bin/vgrind        \
            --with-grap=/usr/bin/grap

配置脚本的含义：

--disable-setuid

此项防止 man 程序 setuid 为 man 用户。

--with-...

这三个参数用于设置一些默认程序。lynx 是一个基于文本的网络浏览器 (安装指令见 BLFS)，vgrind 将程序源代码转换成 Groff 输入，grap 常用于在 Groff 文档中设置图标的变量属性。vgrind 和 grap 程序通常在查看手册页是不需要，而且不在 LFS 或者 BLFS 之内，但是完成 LFS 之后，你应该能够自己安装之。

编译程序包： 

make

要测试结果，执行： 

make check

安装程序包： 

make install

6.55.3. Man-DB 的内容

安装的程序：accessdb、apropos (到 whatis 的链接)、catman、lexgrog、man、mandb、manpath、whatis 和 zsoelim
安装的库：libman.so、libmandb.so
安装的目录：/usr/lib/man-db、/usr/share/doc/man-db

简介

accessdb

转储 whatis 数据库内容为可读格式

apropos

搜索 whatis 数据库并显示包含给定字符串的每个系统命令的简介

catman

创建或者更新预格式化的手册页

lexgrog

显示给定手册页的单行摘要信息

man

格式化并显示请求的手册页

mandb

创建或更新 whatis 数据库

manpath

显示 $MANPATH 或者 (如果没有设置 $MANPATH) 依据 man.conf 和用户的环境选出的合适的搜索路径中的内容

whatis

搜索 whatis 数据库，显示出以独立单词包含给定关键字的所有系统命令的简介

zsoelim

读取文件，将其中的 .so 文件表格替换为文件的内容文件

libman

包含 man 的运行时支持

libmandb

包含 man 的运行时支持
