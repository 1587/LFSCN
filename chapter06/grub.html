6.47. GRUB-2.00

GRUB 程序包包含大一统引导加载器。

预计编制时间：0.7 SBU
需求磁盘空间：112 MB

6.47.1. 安装 GRUB

修复一个该程序包与 Glibc-2.17 间的不兼容错误： 

sed -i -e '/gets is a/d' grub-core/gnulib/stdio.in.h

准备编译 GRUB ： 

./configure --prefix=/usr          \
            --sysconfdir=/etc      \
            --disable-grub-emu-usb \
            --disable-efiemu       \
            --disable-werror

--disable-werror 选项允许编制在更新的 flex 版本警告的情况下仍可以完成。另外的 --disable 开关通过禁用 LFS 不需要的特性和测试程序来使编制的内容最少。

编译程序包: 

make

此程序包不含测试套件。

安装程序包： 

make install

有关使用 GRUB 来使你的 LFS 系统可引导的内容，查阅 第 8.4 节 “使用 GRUB 设置引导过程”.

6.47.2. GRUB 的内容

安装的程序：grub-bios-setup、grub-editenv、grub-fstest、grub-install、grub-kbdcomp、grub-menulst2cfg、grub-mkconfig、grub-mkimage、grub-mklayout、grub-mknetdir、grub-mkpasswd-pbkdf2、grub-mkrelpath、grub-mkrescue、grub-mkstandalone、grub-ofpathname、grub-probe、grub-reboot、grub-script-check、grub-set-default、grub-sparc64-setup
安装的目录：/usr/lib/grub, /etc/grub.d, /usr/share/grub, /boot/grub

简介

grub-bios-setup

是 grub-install 的一个帮助程序

grub-editenv

一个编辑环境块的工具

grub-fstest

调试文件流驱动的工具

grub-install

在你的驱动器上安装 GRUB

grub-kbdcomp

用于转换一个 xkb 布局转换成 GRUB 可以识别的格式

grub-menulst2cfg

转换一个 GRUB Legacy menu.lst 为一个 GRUB 2 可使用的 grub.cfg

grub-mkconfig

生成一个 grub 配置文件

grub-mkimage

制作一个可引导的 GRUB 镜像

grub-mklayout

生成一个 GRUB 键盘布局文件

grub-mknetdir

准备一个 GRUB 网络引导目录

grub-mkpasswd-pbkdf2

生成一个加密的 PBKDF2 密码以在引导目录中使用

grub-mkrelpath

制作一个相对它的根的系统路径名

grub-mkrescue

制作一个软盘或 CDROM/DV　可用的可引导　GRUB　镜像

grub-mkstandalone

生成一个独立镜像

grub-ofpathname

是一个打印出一个　GRUB　驱动器路径的帮助程序

grub-probe

探测给定路径或驱动器的驱动器信息

grub-reboot

仅为下一次引导设置　GRUB　的默认引导入口

grub-script-check

检查　GRUB　配置脚本中的语法错误

grub-set-default

设置 GRUB 的默认驱动入口

grub-sparc64-setup

是一个 grub-setup 的帮助程序
