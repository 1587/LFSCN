6.43. Flex-2.5.37

Flex 程序包包含用于创建识别文本样式的程序的工具。

预计编制时间：0.4 SBU
需求磁盘空间：39 MB

6.43.1. 安装 Flex

首先修复一些回归测试： 

patch -Np1 -i ../flex-2.5.37-bison-2.6.1-1.patch

准备编译 Flex： 

./configure --prefix=/usr             \
            --docdir=/usr/share/doc/flex-2.5.37

编译程序包： 

make

要测试结果 (大约 0.5 SBU)，执行： 

make check

安装程序包： 

make install

有一些程序包会在 /usr/lib 中寻找 lex 库。创建一个符号链接解决此问题： 

ln -sv libfl.a /usr/lib/libl.a

有几个程序还不知道 flex，会试图运行它的前身 lex。要支持这些程序，创建一个打包脚本，命名为 lex，以 lex 模拟模式调用 flex： 

cat > /usr/bin/lex << "EOF"
#!/bin/sh
# Begin /usr/bin/lex

exec /usr/bin/flex -l "$@"

# End /usr/bin/lex
EOF
chmod -v 755 /usr/bin/lex

6.43.2. Flex 的内容

安装的程序：flex, lex++ (到 flex 的链接) 和 lex
安装的库：libfl.a 和 libfl_pic.a
安装的目录：/usr/share/doc/flex-2.5.37

简介

flex

一个工具，用于创建能够识别文本样式的程序；它允许指定样式查找的规则，消除了开发特定程序的需求

flex++

flex 的一个扩展，用于创建 C++ 代码和类。是到 flex 的链接

lex

一个脚本，以 lex 模拟模式运行 flex

libfl.a

flex 库
