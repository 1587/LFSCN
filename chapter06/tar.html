6.59. Tar-1.26

Tar 程序包包含一个归档程序。

预计编制时间: 2.4 SBU
需求磁盘空间: 34 MB

6.59.1. 安装 Tar

修复此程序包与 Glibc-2.16.0 不兼容的问题： 

sed -i -e '/gets is a/d' gnu/stdio.in.h

准备编译 Tar： 

FORCE_UNSAFE_CONFIGURE=1  \
./configure --prefix=/usr \
            --bindir=/bin \
            --libexecdir=/usr/sbin

配置选项的含义：

FORCE_UNSAFE_CONFIGURE=1

此项强制 mknod 的测试以 root 用户运行。此行为通常认为很危险，但由于系统编制尚不完整，可以覆盖此选项。

编译程序包： 

make

要测试结果 (约 1 SBU)，执行： 

make check

安装程序包： 

make install
make -C doc install-html docdir=/usr/share/doc/tar-1.26

6.59.2. Tar 的内容

安装的程序: rmt 和 tar
安装的目录: /usr/share/doc/tar-1.26

简介

rmt

通过跨进程通信连接远程修改磁带驱动器Remotely manipulates a magnetic tape drive through an interprocess communication connection

tar

创建、提取归档文件 (压缩包) 或列出其内容
