6.34. Libtool-2.4.2

Libtool 程序包包含 GNU 通用库支持脚本。它将使用共享库的复杂性打包为统一、可移植的接口。

预计编制时间：3.0 SBU
需求磁盘空间：37 MB

6.34.1. 安装 Libtool

准备编译 Libtool： 

./configure --prefix=/usr

编译程序包： 

make

要测试结果 (大约 3.0 SBU)，执行： 

make check

安装程序包： 

make install

6.34.2. Libtool 的内容

安装的程序：libtool 和 libtoolize
安装的库：libltdl.{a,so}
安装的目录：/usr/include/libltdl、/usr/share/libtool

简介

libtool

提供通用化的库编制支持服务

libtoolize

提供向程序包添加 libtool 支持的标准方式

libltdl

隐藏 dlopen 库的困难
