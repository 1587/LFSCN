6.39. Automake-1.12.4

Automake 程序包包含创建用于 Autoconf 的 Makefile 的程序。

预计编制时间ï¼少于 0.1 SBU (含测试 34.1 SBU)
需求磁盘空间ï¼100 MB

6.39.1. 安装 Automake

准备编译 Automake： 

./configure --prefix=/usr --docdir=/usr/share/doc/automake-1.12.4

编译程序包： 

make

注意

测试会占用很长时间，超过 30 SBU。不推荐运行之。

要测试结果，执行： 

make check

安装程序包： 

make install

6.39.2. Automake 的内容

安装的程序ï¼acinstall、aclocal、aclocal-1.12、automake、automake-1.12、compile、config.guess、config.sub、depcomp、elisp-comp、install-sh、mdate-sh、missing、mkinstalldirs、py-compile、symlink-tree 和 ylwrap
安装的目录ï¼/usr/share/aclocal-1.12、/usr/share/automake-1.12、/usr/share/doc/automake-1.12.4

简介

acinstall

一个脚本，安装 aclocal 样式的 M4 文件

aclocal

基于 configure.in 文件的内容创建 aclocal.m4 文件

aclocal-1.12

到 aclocal 的硬链接

automake

一个用于以 Makefile.am 文件创建 Makefile.in 文件的个工具；要创建一个程序包的所有 Makefile.in 文件，在顶级目录运行此程序；通过扫描 configure.in 文件，它自动找到每个对应的 Makefile.am 文件并创建相应的 Makefile.in 文件

automake-1.12

到 automake 的硬链接

compile

编译器的打包

config.guess

一个脚本，试图猜测给定的编制、宿主或者目标架构的规范化三联字串

config.sub

一个配置校验子途径脚本

depcomp

一个脚本，用于编译一个程序并创建依赖关系信息附加到指定的输出

elisp-comp

字节编译 Emacs Lisp 代码

install-sh

一个脚本，安装程序、脚本或者数据文件

mdate-sh

一个脚本，打印出文件或者目录的修改时间

missing

一个脚本，安装时作为缺失的 GNU 程序的通用存根使用

mkinstalldirs

一个脚本，创建目录树

py-compile

编译 Python 程序

symlink-tree

一个脚本，创建一个目录树的符号链接树

ylwrap

lex 和 yacc 的一个打包
