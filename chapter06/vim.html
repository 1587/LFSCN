6.62. Vim-7.3

Vim 程序包包含一个强大的文本编辑器。

预计编制时间：1.1 SBU
需求磁盘空间：96 MB

Vim 的替代品

如果你喜欢其它的编辑器——例如 Emacs、Joe 或者 Nano——请参考 http://www.linuxfromscratch.org/blfs/view/svn/postlfs/editors.html，其中包含推荐的安装指令。

6.62.1. 安装 Vim

首先改变 vimrc 配置文件的默认位置到 /etc： 

echo '#define SYS_VIMRC_FILE "/etc/vimrc"' >> src/feature.h

现在准备编译 Vim： 

./configure --prefix=/usr --enable-multibyte

配置选项的含义：

--enable-multibyte

此开关启用编辑多字节字符编码文件的支持。如果使用了带有多字节字符集的区域数据就需要开启。此开关对于编辑由 Fedora 等默认使用 UTF-8 字符集的 Linux 发行版创建的文本文件也有用。

编译程序包： 

make

要测试结果，执行： 

make test

然而，此测试套件会向屏幕输出大量二进制数据，可能导致当前终端的设置出现问题。可以将输出重定向到一个日志文件来解决。测试成功会输出 “ALL DONE”。

安装程序包： 

make install

很多用户习惯于使用 vi 而不是 vim。要允许在用户按习惯输入 vi 时运行 vim，创建其二进制和提供的手册页语言的符号链接： 

ln -sv vim /usr/bin/vi
for L in  /usr/share/man/{,*/}man1/vim.1; do
    ln -sv vim.1 $(dirname $L)/vi.1
done

Vim 的文档默认安装于 /usr/share/vim。以下符号链接允许通过 /usr/share/doc/vim-7.3 访问之，使其与其他程序包的手册位置一致： 

ln -sv ../vim/vim73/doc /usr/share/doc/vim-7.3

如果要在 LFS 系统上安装 X 窗口系统，安装过后可能需要重新编译 Vim。Vim 编辑器带有一个图形界面版本，需求 X 和一些额外的库。有关此过程，参考 BLFS 中的 http://www.linuxfromscratch.org/blfs/view/svn/postlfs/editors.html#postlfs-editors-vim。

6.62.2. 配置 Vim

vim 默认以不兼容 vi 的模式运行。这对于过去使用过其他编辑器的用户来说可能很陌生。以下包括了 “不兼容的” 设置，以突出使用了新的行为这一事实。它也提醒想要使用 “兼容” 模式的用户在配置文件起始处设置之。这非常必要，因为他会改变其他设置，而覆盖必须在此之后。运行以下命令创建一个默认的 vim 配置文件： 

cat > /etc/vimrc << "EOF"
" Begin /etc/vimrc

set nocompatible
set backspace=2
syntax on
if (&term == "iterm") || (&term == "putty")
  set background=dark
endif

" End /etc/vimrc
EOF

set nocompatible 设置使 vim 以更通用的方式 (默认) 而不是 vi 兼容模式运行。移除 “no” 以保持旧的 vi 行为。set backspace=2 设置允许跨行退格、自动回车和开始插入。syntax on 参数启用 vim 的语法高亮。最后 if 声明和 set background=dark 设置修正 vim 对某些终端模拟器背景颜色的猜测。此项对于这些程序上的黑背景能给出更好的高亮色彩方案。

其他可用选项的文档可以通过运行以下命令获取： 

vim -c ':options'

注意

Vim 默认只安装英语的拼写文件。要安装其他语言的拼写文件，到 ftp://ftp.vim.org/pub/vim/runtime/spell/ 下载你的语言和字符编码的 *.spl，还可以包括 *.sug 文件并保存到 /usr/share/vim/vim73/spell/ 中。

要使用这些拼写文件需要 /etc/vimrc 中的一些配置，例如： 

set spelllang=en,ru
set spell

更多信息位于对应的 README 和以上地址中。

6.62.3. Vim 的内容

安装的程序： ex (到 vim 的链接)、rview (到 vim 的链接)、rvim (到 vim 的链接)、vi (到 vim 的链接)、view (到 vim 的链接)、vim、vimdiff (到 vim 的链接)、vimtutor 和 xxd
安装的库：/usr/share/vim

简介

ex

以 vim ex 模式启动 vim

rview

是 view 的一个受限制版本；不能启动 shell 命令，view 也不能挂起

rvim

是 vim 的一个受限制版本；不能启动 shell 命令，vim 也不能挂起

vi

到 vim 的链接

view

以制度模式启动 vim

vim

编辑器

vimdiff

以 vim 编辑一个文件的两个或者三个版本并显示差异

vimtutor

讲解 vim 的基本按键和命令

xxd

创建给定文件的十六进制转储或反之，可以用于二进制补丁
