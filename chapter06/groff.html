6.45. Groff-1.21

Groff 程序包包含用于处理和格式化文本的程序。

预计编制时间ï¼0.5 SBU
需求磁盘空间ï¼83 MB

6.45.1. 安装 Groff

Groff 期待环境变量 PAGE 包含默认纸张尺寸。对于美国用户适用的是 PAGE=letter，而在其他地方 PAGE=A4 会更合适。默认纸张尺寸虽然是在编译时配置，此后也可以将 “A4” 或 “letter” echo 到 /etc/papersize 文件来覆盖。

准备编译 Groff： 

PAGE=<paper_size> ./configure --prefix=/usr

编译程序包： 

make

此程序包不包含测试套件。

安装程序包： 

make install

某些文档程序，例如 xman，需要以下符号链接才能工作： 

ln -sv eqn /usr/bin/geqn
ln -sv tbl /usr/bin/gtbl

6.45.2. Groff 的内容

安装的程序ï¼addftinfo、afmtodit、chem、eqn、eqn2graph、gdiffmk、geqn (到 eqn 的链接)、grap2graph、grn、grodvi、groff、groffer、grog、grolbp、grolj4、grops、grotty、gtbl (到 tbl 的链接)、hpftodit、indxbib、lkbib、lookbib、mmroff、neqn、nroff、pdfroff、pfbtops、pic、pic2graph、post-grohtml、preconv、pre-grohtml、refer、roff2dvi、roff2html、roff2pdf、roff2ps、roff2text、roff2x、soelim、tbl、tfmtodit 和 troff
安装的目录ï¼/usr/lib/groff、/usr/share/doc/groff-1.21、/usr/share/groff

简介

addftinfo

读取一个 troff 字体文件并添加额外的字号信息供 groff 系统使用

afmtodit

创建一个字体文件供 groff 和 grops 使用

chem

Groff 预处理器，用于产生化学结构式

eqn

将 troff 输入文件中嵌入的等式描述编译为 troff 能够理解的命令

eqn2graph

将 troff EQN (等式) 转换成裁剪过的图像

gdiffmk

标记出 groff/nroff/troff 文件之间的差异

geqn

到 eqn 的链接

grap2graph

将 grap 图标转换成裁剪过的位图

grn

一个 groff 预处理器，用于 gremlin 文件

grodvi

groff 的一个驱动，产生 TeX dvi 格式的输出

groff

groff 文档格式化系统的一个前端；通常会运行 troff 程序和选中设备对应的预处理器

groffer

在 X 和 tty 终端上显示 groff 文件和手册页

grog

读取文件，猜测打印文件需要 groff 选项中的 -e、-man、-me、-mm、-ms、-p、-s 和 -t 哪些，并报告包含这些选项的 groff 命令

grolbp

是一个 groff 驱动，用于佳能 CAPSL 打印机 (LBP-4 和 LBP-8 系列激光打印机)

grolj4

是一个 groff 驱动，提供适用于惠普 LaserJet 4 打印机的 PCL5 格式输出

grops

将 GNU troff 的输出翻译成 PostScript

grotty

将 GNU troff 的输出翻译成适用于类打字机设备的格式

gtbl

到 tbl 的链接

hpftodit

以惠普字号文件创建一个字体文件，供与 groff -Tlj4 一同使用

indxbib

给参考文献数据库创建一个反转索引，带有一个特殊指定的文件供与 refer、lookbib 和 lkbib 一同使用

lkbib

搜索参考文献数据库，查找包含指定键值的提及

lookbib

在标准出错上打印一个提示符 (除非标准输出不是终端)，从标准输入读取一行关键字，搜索参考文献数据库的指定文件，查找这些关键字并且在标准输出上打印结果，重复此过程直到输入结束为止。

mmroff

groff 的一个简单预处理器

neqn

针对 ASCII 输出格式化等式

nroff

一个脚本，以 groff 模拟 nroff 命令

pdfroff

用 groff 创建 pdf 文档

pfbtops

将 .pfb 格式的 PostScript 字体翻译成 ASCII

pic

将嵌入于 troff 或者 TeX 输入文件中的图片描述翻译成 TeX 或者 troff 能够理解的命令

pic2graph

将 PIC 图表转换成剪裁过的图像

post-grohtml

将 GNU troff 的输出翻译成 HTML

preconv

将输入文件的转换为 troff 识别的编码

pre-grohtml

将 GNU troff 的输出翻译成 HTML

refer

将一个文件的内容复制至标准输出，其中 .[ 和 .] 之间的内容作为引文翻译，.R1 和 .R2 之间的内容翻译成处理引用的命令

roff2dvi

将 roff 文件翻译成 DVI 文件

roff2html

将 roff 文件翻译成 HTML 文件

roff2pdf

将 roff 文件翻译成 PDF 文件

roff2ps

将 roff 文件翻译成 PS 文件

roff2text

将 roff 文件翻译成文本文件

roff2x

将 roff 文件翻译成其他格式

soelim

读取文件，将其中的 .so 文件表格替换为文件的内容文件

tbl

将 troff 输入文件中嵌入的表格描述编译成 troff 理解的命令

tfmtodit

创建一个字体文件供与 groff -Tdvi 共同使用

troff

与 Unix 的 troff 高度兼容；它通常应该使用 groff 命令调用，此法还会按正确的顺序和正确的选项运行预处理器和后处理器
