5.21. Findutils-4.4.2

Findutils 程序包包含用于查找文件的程序。这些程序能够递归搜索目录树并创建、维护和搜索一个数据库 (通常比递归搜索快，但如果数据库最近没有更新可能会不不可靠)。

预计编制时间: 0.3 SBU
需求磁盘空间: 20 MB

5.21.1. 安装 Findutils

准备编译 Findutils： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Findutils 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.40.2 节 “Findutils 的内容”。
