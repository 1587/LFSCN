5.27. Patch-2.7.1

Patch 程序包包含一个应用由 diff 程序产生的 “patch” 文件 (补丁) 从而修改或创建文件的程序。

预计编制时间: 0.1 SBU
需求磁盘空间: 3.4 MB

5.27.1. 安装 Patch

准备编译 Patch： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Patch 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.56.2 节 “Patch 的内容”。
