5.23. Grep-2.12

The Grep package contains programs for searching through files.

预计编制时间: 0.2 SBU
需求磁盘空间: 18 MB

5.23.1. 安装 Grep

准备编译 Grep： 

./configure --prefix=/tools --disable-perl-regexp

配置开关的含义：

--disable-perl-regexp

此项确保 grep 程序不会链接至宿主上可能出现但进入 chroot 环境就不再可用的 Perl 兼容正则表达式 (PCRE) 库。

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Grep 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.29.2 节 “Contents of Grep”。
