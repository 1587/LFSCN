5.19. Diffutils-3.2

Diffutils 程序包包含用于显示文件或目录之间的差别的程序。

预计编制时间: 0.1 SBU
需求磁盘空间: 6.1 MB

5.19.1. 安装 Diffutils

准备编译 Diffutils： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Diffutils 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.38.2 节 “Diffutils 的内容”。
