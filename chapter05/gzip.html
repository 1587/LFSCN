5.24. Gzip-1.5

Gzip 程序包包含用于压缩和解压缩文件的程序。

预计编制时间：0.2 SBU
需求磁盘空间：10 MB

5.24.1. 安装 Gzip

准备编译 Gzip： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Gzip 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于 第 6.49.2 节 “Gzip 的内容”。
