5.12. DejaGNU-1.5

DejaGNU 程序包包含一套框架用于测试其他程序。

预计编制时间: 少于 0.1 SBU
需求磁盘空间: 4.1 MB

5.12.1. 安装 DejaGNU

准备编译 DejaGNU： 

./configure --prefix=/tools

编制安装程序包： 

make install

要测试结果，执行： 

make check

5.12.2. DejaGNU 的内容

安装的程序: runtest

简介

runtest

一个包装脚本，定位正确的 expect shell 然后运行 DejaGNU
