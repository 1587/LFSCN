7.14. 创建 /etc/inputrc 文件

inputrc 文件处理指定情况下的键盘映射。此文件是 Readline——有关输入的库——的起始文件，被 Bash 和多数其他 shell 使用。

很多人不需要用户指定键盘映射，所以以下命令创建全局的 /etc/inputrc 文件供多有登录的用户使用。如果你此后决定要基于单一用户覆盖此默认值，可以在用户的主目录中创建一个 .inputrc 文件，其中包含修改过的映射。

有关如何编辑 inputrc 文件的更多信息参见 info bash 中的 Readline Init File 一节。info readline 也能提供一些信息。

以下为全局通用的 inputrc 文件，带有注释，解释了不通选项的功能。注意这些注释不能与命令位于同一行。运行一下命令创建此文件： 

cat > /etc/inputrc << "EOF"
# Begin /etc/inputrc
# Modified by Chris Lynn <roryo@roryo.dynup.net>

# Allow the command prompt to wrap to the next line
set horizontal-scroll-mode Off

# Enable 8bit input
set meta-flag On
set input-meta On

# Turns off 8th bit stripping
set convert-meta Off

# Keep the 8th bit for display
set output-meta On

# none, visible or audible
set bell-style none

# All of the following map the escape sequence of the value
# contained in the 1st argument to the readline specific functions
"\eOd": backward-word
"\eOc": forward-word

# for linux console
"\e[1~": beginning-of-line
"\e[4~": end-of-line
"\e[5~": beginning-of-history
"\e[6~": end-of-history
"\e[3~": delete-char
"\e[2~": quoted-insert

# for xterm
"\eOH": beginning-of-line
"\eOF": end-of-line

# for Konsole
"\e[H": beginning-of-line
"\e[F": end-of-line

# End /etc/inputrc
EOF
