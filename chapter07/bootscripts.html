7.6. LFS-Bootscripts-20120229

The LFS-Bootscripts package contains a set of scripts to start/stop the LFS system at bootup/shutdown.

预计编制时间: less than 0.1 SBU
需求磁盘空间: 260 KB

7.6.1. 安装 LFS-Bootscripts

安装这个包： 

make install

7.6.2. LFS-Bootscripts 的内容

被安装的脚本: checkfs, cleanfs, console, functions, halt, ifdown, ifup, localnet, modules, mountfs, mountkernfs, network, rc, reboot, sendsignals, setclock, static, swap, sysctl, sysklogd, template, udev, and udev_retry
安装文件夹: /etc/rc.d, /etc/init.d (符号链接), /etc/sysconfig, /lib/services, /lib/lsb (符号链接)

简介

checkfs

Checks the integrity of the file systems before they are mounted (with the exception of journal and network based file systems)

cleanfs

Removes files that should not be preserved between reboots, such as those in /var/run/ and /var/lock/; it re-creates /var/run/utmp and removes the possibly present /etc/nologin, /fastboot, and /forcefsck files

console

Loads the correct keymap table for the desired keyboard layout; it also sets the screen font

functions

Contains common functions, such as error and status checking, that are used by several bootscripts

halt

Halts the system

ifdown

Stops a network device

ifup

Initializes a network device

localnet

Sets up the system's hostname and local loopback device

modules

加载由文件 /etc/sysconfig/modules 指定的内核模块。

mountfs

挂载所有的文件系统，除非它被标记为 noauto 或者这个文件系统跟网络相关。

mountkernfs

挂载内核的虚拟文件系统，例如 proc 文件系统。

network

设置网络接口，例如网卡，并设置默认网关。

rc

运行级别（run-level）控制脚本。会将根据符号链接的名字决定的顺序，一个一个地执行其他启动脚本脚本。

reboot

重启系统

sendsignals

确保所有的进程在关机或重启前被结束。

setclock

当硬件时钟时间没有设置成UTC时将内核时间设置为本地时间。

static

提供给一个网络接口分配静态IP地址时需要的功能。

swap

启用或停用交换（swap）分区或文件。

sysctl

如果 /etc/sysctl.conf 文件存在，将其中的配置文件加载进内核。

sysklogd

启动和停止系统及内核的日志守护进程。

template

一个用来给其他守护进程创建启动脚本的模板。

udev

准备 /dev 文件夹，并启动Udev

udev_retry

重试失败的Udev事件（uevent），根据需要将生成的规则（rule）文件复制到 /etc/udev/rules.d 文件夹。
